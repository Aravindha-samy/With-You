Identity-First AI Architecture 

“With You – When memory fades, presence remains.” 

With You is not a reminder app. 
It is a persistent emotional memory companion. 

It protects:	 

Identity 

Relationships 

Emotional safety 

Personal story continuity 

The system works as a Cognitive Mesh — multiple intelligent agents working together quietly in the background. 

Shape 

🌿 The with You Cognitive Mesh 

Instead of one chatbot, we use specialized emotional agents. 

Each one has a clear responsibility. 

Shape 

🌞 1. Aurora – The Orchestrator 

Role: The central coordinator. 

Aurora decides: 

What the patient is asking 

Which agent should respond 

Whether the patient sounds anxious 

If caregiver alert is needed 

Think of Aurora as: 

The calm traffic controller of the entire system. 

It does not store memory. 
It directs intelligence. 

Shape 

🏠 2. Harbor – The Orientation Agent 

Role: Keeps the patient grounded. 

Harbor answers: 

What day is it? 

Where am I? 

Who is visiting today? 

What is happening next? 

It also: 

Handles repeated questions gently 

Increases reassurance if confusion rises 

Harbor protects: 

Stability and safety. 

Shape 

👨‍👩‍👧 3. Roots – Identity & Relationship Agent 

Role: Protects relational identity. 

Roots answers: 

Who is this? 

Who is Anna? 

How am I related to this person? 

It maintains: 

Family structure 

Emotional importance of relationships 

Shared memories 

Roots protects: 

Familiarity and belonging. 

Shape 

💬 4. Solace – Emotional Intelligence Agent 

Role: Detects and regulates emotional distress. 

Solace monitors: 

Tone of voice 

Word patterns 

Anxiety signals 

Repetition frequency 

If distress is detected, Solace: 

Calms the patient 

Activates Calm Mode 

Alerts caregiver if needed 

Solace protects: 

Emotional safety and dignity. 

Shape 

🧠 5. Echo – Memory Layer Agent 

Role: Long-term memory preservation. 

Echo: 

Stores conversation history 

Tracks emotional trends 

Detects cognitive patterns 

Identifies increasing repetition 

Echo enables: 

Predictive anxiety detection 

Longitudinal emotional modeling 

Echo protects: 

Continuity over time. 

Shape 

👩‍⚕️ 6. Guardian – Caregiver Agent 

Role: The caregiver co-pilot. 

Guardian provides: 

Daily cognitive summary 

Emotional trend insights 

Escalation alerts 

Simple analytics dashboard 

Guardian helps caregivers: 

Act early instead of reacting late. 

Shape 

🌅 7. Legacy – Story Continuity Agent 

Role: Maintains life narrative. 

If the patient says: 
“I used to work…” 

Legacy gently reinforces: 
“You worked as an engineer for 25 years.” 

Legacy prevents: 

Identity erosion. 

This is one of the most radical differentiators. 

Shape 

🌿 Two Operating Modes (Simple) 

The system runs in two modes only. 

Shape 

MODE A – Structured Navigation (Button Mode) 

Used when the patient taps predefined buttons. 

Example: 
“Family Photos” 

Flow: 

Patient taps button 

Aurora routes directly to the correct agent 

Agent retrieves stored information 

Response shown 

No advanced AI reasoning needed. 

This mode is: 

Fast 

Stable 

Controlled 

Low risk 

Best for moderate cognitive stages. 

Shape 

MODE B – Free Speech Mode 

Used when patient speaks naturally. 

Example: 
“Is Anna coming today?” 

Flow: 

Voice converted to text 

Aurora analyzes meaning and emotion 

Aurora selects the correct agent 

Agent retrieves memory 

Response delivered (optionally spoken back) 

This mode enables: 

Emotional understanding 

Anxiety detection 

Intelligent routing 

Context awareness 

This is where predictive intelligence operates.
